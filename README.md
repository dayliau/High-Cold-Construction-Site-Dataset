项目名称：基于多模态多Agent协同机制的工地智能监管系统 - 开源数据集

## 1. 数据集简介
针对高寒地区（典型极端低温、冻融循环）施工现场安全监管需求，
采集并构建了包含视频（视觉）、音频（声学）、结构应力与气象（环境传感器）的多模态数据集，
旨在为多Agent协同架构下的异常检测提供基准算法测试环境。

## 2. 数据格式与多模态对齐说明
为便于学术研究与算法部署，项目组对原始异构数据进行了【场景级与文件名级（File-level）的时空重构与粗对齐】：
- **时空同步**：所有多模态数据均按特定安全事件（Event-driven）进行切片包装，确保在同一事件文件夹下的各模态数据具有时序一致性。
- **视觉模态**：1080P/30fps H.264 视频，记录人员行为及结构表面异变。
- **声学模态**：16kHz 单声道 WAV 音频，记录高寒冻裂、机械异常轰鸣。
- **环境模态**：包含温度、湿度及结构应力形变的综合时间序列 CSV 文件。
